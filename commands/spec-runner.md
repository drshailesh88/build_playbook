# spec-runner — DEPRECATED

Replaced by `/playbook:qa-run` + `/playbook:author-locked-tests`.

The new pipeline generates tests from contracts under source-denied
permissions (blueprint A2/T2) and runs them through the hardened gate chain
with diff auditing. Tests generated by `spec-runner` had no enforcement
layer — agents could weaken them during repair.

## Use instead

```
/playbook:contract-pack <feature>      # Interactive contract authoring
/playbook:author-locked-tests <feature> # Generate tests from contract (source denied)
/playbook:wire-selectors <feature>      # Tight-scope selector fix-up
/playbook:qa-run                        # Run the full QA session
```

Historical version preserved at `commands/deprecated/spec-runner.md`.
